# Monorepo Fullstack App

Fullstack chat application with FastAPI backend and placeholder frontend.

## Structure

- `backend/` - FastAPI + SQLAlchemy + OpenAI
- `frontend/` - Frontend (placeholder)

## Running

```bash
cd backend
pip install -r requirements.txt
python main.py
```

## Environment

Set up `.env` in backend/ with DATABASE_URL and OPENAI_API_KEY.
